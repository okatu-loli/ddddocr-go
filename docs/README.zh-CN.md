# ddddocr-go 中文文档

`ddddocr-go` 是基于本地 ONNX 模型的 Go SDK 和 CLI，用于 OCR、目标检测、点选验证码点击顺序识别，以及简单滑块匹配。

## 来源说明

本项目的原始模型和代码实现参考来自 [sml2h3/ddddocr](https://github.com/sml2h3/ddddocr)。上游项目是 MIT 协议的 Python 通用验证码识别 SDK，由 `sml2h3` 与 `kerlomz` 共同开发。
